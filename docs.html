üß© Basic Usage

Create a scroll section and define data attributes:

<section class="scroll-section"
  data-folder="frames"
  data-name="frame_"
  data-frames="150"
  data-ext="jpg"
  data-length="4">
  <canvas></canvas>
</section>

Now simply initialize ScrollFX ‚Äî it automatically reads your attributes and syncs animation with scroll.
