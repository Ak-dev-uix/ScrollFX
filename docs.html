  data-scrollfx
  data-folder="shell"

  data-frames="636"

‚öôÔ∏è Available Attributes

Attribute Description
data-scrollfxRequired ‚Äî identifies a section for ScrollFX initialization.
data-folderPath to your frames folder (e.g. shell).
data-nameBase filename prefix (e.g. shell- ‚Üí shell-001.webp).
data-framesTotal number of frames to animate through.
data-extFrame image format (e.g. jpg, png, webp).
data-lengthZero-padding length for filenames (e.g. 4 ‚Üí 0001).
data-bgOptional fallback background image shown before/after animation.
data-sectionOptional unique ID for debugging or multiple ScrollFX sections.

‚ú® Features

  • Pure JavaScript + GSAP based, no frameworks
  • Responsive and auto-resizing canvas
  • Supports JPG, PNG, WebP sequences
  • Scroll-linked animation using GSAP ScrollTrigger
  • Zero setup ‚Äî just data attributes!

üí° Pro Tips

  • Combine with GSAP timelines for overlay text or parallax.
